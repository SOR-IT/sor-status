# Studio OntheRock — Program Status

Public status feed for two ongoing technology programs at Studio OntheRock, Honolulu.

**Status page:** https://sor-it.github.io/sor-status/

---

## What this repo contains

- `docs/STATUS.md` — current milestone status, updated on each program checkpoint
- `docs/index.html` — self-updating dashboard that reads STATUS.md at page load

## What this repo does not contain

Implementation detail, architecture, code, or process documentation.
That material is maintained separately in a private repository.

## How status updates work

`docs/STATUS.md` is updated at each program checkpoint and committed here.
The dashboard reads the file live from GitHub on every page load — no manual
dashboard maintenance required.
